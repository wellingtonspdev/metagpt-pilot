# Estudos e Casos Reais

Esta pasta centraliza apenas a documentacao produzida durante o estudo e a pilotagem do MetaGPT. Ela nao inclui configuracoes `config2.yaml`, chaves de API, logs brutos de containers, snapshots de workspace ou dados de projetos.

## Organizacao

- `research/`: viabilidade de OpenRouter, pesquisa sobre otimizacao de tokens e sintese aplicavel ao MetaGPT.
- `case-studies/`: evidencias da pilotagem, limites de saida estruturada e validacoes feitas durante um projeto real.

Os documentos de caso podem citar um projeto de origem apenas para explicar o experimento. Eles nao definem requisitos obrigatorios para quem usar a skill em outro contexto.

## Como contribuir

Adicione somente experimentos reproduziveis, com contexto, evidencias, conclusao e limite conhecido. Antes de versionar, execute a varredura de segredos descrita no README raiz.
